# Engenharia de Sistemas Agrícolas

Este repositório reúne materiais complementares do livro **Engenharia de Sistemas Agrícolas**, com ênfase em modelagem, simulação e otimização de sistemas aplicados à engenharia agrícola.

O conteúdo inclui algoritmos, dados de exemplo, planilhas e figuras utilizadas ao longo dos capítulos, com foco na aplicação prática e no apoio ao ensino.

## 📘 Conteúdo

Este repositório acompanha o desenvolvimento do livro, incluindo:

- Modelagem de sistemas agrícolas  
- Sistemas mecanizados de produção  
- Determinação de dias disponíveis para operações mecanizadas

- Análise de desempenho operacional  
- Aplicações de probabilidade e cadeias de Markov
